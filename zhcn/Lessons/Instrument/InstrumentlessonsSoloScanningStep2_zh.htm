单人飞行： 仪表扫描
立即飞行此课程

——作者：Rod Machado

三步仪表扫描的步骤2
径向扫描主要仪表

步骤1、步骤2、步骤3：这听起来不像弗雷德·阿斯泰尔的舞蹈入门课程吗？嗯，仪表扫描最好通过易于管理的步骤来教授和理解。从这个意义上说，您的舞伴是仪表板，您的眼睛以有组织的方式从一个仪表舞到另一个仪表。您已经学会了三步扫描的步骤1，现在进入步骤2。

步骤1允许您仅通过参考姿态指示器（AI）将飞机置于任何姿态。然而，仅使用AI作为姿态控制的唯一手段，就像使用热核武器杀死您公寓中的跳蚤一样。它有效，但缺乏邻居们期望的杀虫剂精度。作为仪表飞行员，您需要精确定位头、高度和空速。一旦您在步骤1中选择了新的姿态，您将继续进行步骤2，在那里您将径向扫描主要仪表并微调步骤1中选择的姿态。

步骤2的目的是查看一个或多个飞行仪表，然后对俯仰、坡度或功率进行任何必要的更改，以获得所需飞行姿态。这反过来又允许您获得所需的精确航向、空速和高度。径向扫描术语表示您的扫描从AI开始，到面板上的主要仪表，然后返回AI。扫描模式是沿着相同路径向外然后返回。将您的眼睛路径视为从自行车轮的中心开始，沿着辐条向外和向内辐射，如图1所示。


图1

任何主要的东西都很重要。主要仪表为您提供俯仰、坡度和功率精确控制的最重要信息。您选择的每种姿态都使用三个主要仪表：一个用于俯仰，一个用于坡度，一个用于功率。但是您如何知道这些仪表是什么？毕竟，您有几个可以选择。要回答这个问题，让我们去吃个汉堡。

仪表名称

当您从汉堡宫订购食物时，服务员会按下一个有您选择的物品图片的按钮。点一瓶苏打水，他就会按一个有苏打水的按钮。这种简洁的视觉方法释放了服务员的思维，让他们可以思考更重要的事情，比如哲学、伦理学和费马大定理的另一种证明。当然，如果您说"天气真好"，服务员可能会说"抱歉，我没有那个按钮。"让我们使用类似的标签系统来标识您面板上的主要仪表。

我希望您将图2中所示的标签直接放在屏幕上每个显示仪表的下方（目前我们不会标记VSI）。使用黄色便签纸粘性端的一小段四分之一英寸条。不要使用带有永久性粘合剂的标签（如果您这样做，那个汉堡店总是有您的工作机会！）。


图2
识别主要仪表

图2标识了任何给定飞行条件的主要仪表。假设您刚刚选择了平直飞行的姿态。您应该径向扫描哪些主要仪表？查看面板并找到标记为平直（航向指示器）和水平（高度表）的仪表。航向指示器帮助您直线飞行；高度表帮助您水平飞行；转速表显示选定的功率设置。换句话说，您只能通过扫描这三个仪表来微调平直飞行的姿态。简单，对吧？

假设您已将飞机置于直线爬升（或下降）的姿态。您应该径向扫描哪些主要仪表？找到标记为平直（航向指示器）和爬升（空速表）的仪表。航向指示器帮助您直线飞行；空速表帮助您确定爬升（或下降）的适当俯仰；转速表显示选定的功率设置。

最后，假设您将飞机置于水平转弯的姿态。您应该径向扫描哪些主要仪表？找到标记为水平（高度表）和转弯（转弯协调器）的仪表。高度表帮助您水平飞行；转弯协调器帮助确定所需转弯的坡度量（您很快就会看到如何做到这一点）；转速表显示选定的功率设置。

现在您知道如何确定任何飞行条件要扫描哪些仪表了。接下来，您想要径向扫描主要仪表并观察其指示或检测指针移动。然后，返回AI并进行姿态调整（如果需要）以稳定主要仪表。让我们首先检查我们如何对平直飞行进行此操作。我们假设您刚从另一种姿态进入平直飞行。

径向扫描的基础

除了平直飞行的主要仪表外，图3中的所有仪表都被涂黑了，就像在真正的仪表训练中一样。我们假设您刚刚完成步骤1并将飞机置于平直飞行的姿态。开始步骤2，径向扫描主要仪表并在AI上进行姿态调整（如果需要）以实现精确的平直飞行。在我们继续讨论之前，让我们更多地谈谈径向扫描。


图3

AI下有"开始"字样，因为这是所有径向扫描开始的地方。像自行车轮的轮毂一样，您的扫描将从这里开始，沿着辐条向外辐射到主要仪表。您将在主要仪表上花费大约1到2秒，同时检查任何偏差或指针移动。然后，您将返回AI并进行修正（如果需要）以稳定主要仪表。

当然，您可以径向扫描多个仪表。要做到这一点，您从AI开始，到主要仪表，然后返回AI。从那里，您到另一个主要仪表，然后返回AI，重复该过程，始终返回AI。

例如，在平直飞行中，您将径向扫描航向指示器（平直）、高度表（水平）和转速表（功率）。从AI开始，向下移动到航向指示器。寻找与所需航向的任何偏差。返回AI，进行小坡度更改（如果需要）以停止航向变化或将飞机返回适当航向。从那里，移动到高度表并寻找与所需高度的任何偏差。返回AI，进行小俯仰更改（如果需要）以停止指针或将其返回到适当位置。转速表最后径向扫描。查看转速表并进行最终设置调整（如果需要），然后立即将返回AI。通常，在主要姿态改变期间，转速表不需要径向扫描超过一次。现在，从航向指示器开始，重复径向扫描，直到两个仪表（航向指示器和高度表）指示平直飞行。以下是序列应该看起来的样子。

平直飞行
  1. 从AI开始，径向扫描航向指示器。
  2. 返回AI，并调整坡度（如果需要）以保持270度。
  3. 径向扫描高度表。
  4. 返回AI，并调整俯仰（如果需要）以保持4000英尺。
  5. 径向扫描转速表，并调整油门位置（如果需要）以获得2300转/分钟的巡航RPM（通常不需要再次径向扫描转速表）。
  6. 继续径向扫描航向指示器和高度表，进行小姿态修正，直到飞机建立平直飞行。

径向扫描的秘诀是快速扫描每个主要仪表至少一次，然后在特别径向扫描任何特定仪表之前花额外时间。这使您能够确定飞机与所需姿态的接近程度，并让您了解稳定飞机需要多少工作。让我们检查一下我们如何径向扫描直线爬升的主要仪表。

直线爬升

再次，除了直线爬升的主要仪表外，所有仪表都被涂黑了（图4）。假设您刚进入直线爬升并正在开始三步扫描的步骤2。您将调整AI上的姿态，以80节精确爬升，航向270度。


图4

以下是您用来径向扫描主要仪表的序列。

  1. 从AI开始，径向扫描航向指示器。
  2. 返回AI，并调整坡度（如果需要）以保持270度航向。
  3. 径向扫描空速表。
  4. 返回AI，并调整俯仰（如果需要）以保持80节。
  5. 径向扫描转速表（如果需要），并将RPM调整到2400转/分钟的爬升值（不需要再次径向扫描转速表）。
  6. 继续径向扫描航向指示器和空速表，进行小姿态修正，直到飞机建立在270度航向上以80节精确爬升。

就是这样。提前知道要查看哪些仪表以精确控制您的姿态，消除了仪表飞行的神秘感。

现在让我们尝试对我们扫描的步骤2进行水平飞行转弯。

水平飞行转弯

除了水平转弯的主要仪表外，所有仪表都被涂黑了（图5）。


图5

假设您刚在4000英尺进入左水平转弯，正在开始步骤2。您应该开始径向扫描主要仪表，并调整AI上的姿态，以精确高度4000英尺和标准转弯率转弯。

呃？什么是标准转弯率？

标准转弯率允许飞机以每秒3度的速率改变航向。在之前的课程中，我建议您以20度坡度转弯。这完全没问题，但为了更高的精度，我希望您以标准转弯率进行。您将通过调整坡度直到转弯协调器飞机的机翼位于第二个白色索引标记上，如图5所示。

现在飞机将以每秒精确3度的速率改变航向。标准转弯率让您知道完成转弯需要多长时间。毕竟，以每秒3度的速率，完成360度转弯需要两分钟，完成180度转弯需要一分钟。以下是您用来径向扫描主要仪表的序列。

  1. 从AI开始，径向扫描高度表。
  2. 返回AI，并调整俯仰（如果需要）以保持4000英尺。
  3. 径向扫描转弯协调器。
  4. 返回AI，并调整坡度（如果需要）以保持标准转弯率。
  5. 径向扫描转速表（如果需要），并将RPM调整到2300转/分钟的巡航值（不需要再次径向扫描转速表）。
  6. 继续径向扫描高度表和转弯协调器，进行小姿态修正，直到飞机在4000英尺水平飞行中以标准转弯率左转。

如果您已经走到这一步，您很快就有资格说："谁是最好的飞行员……为什么是我？"在我们的最后一个例子中，我们将检查如何应用步骤2来完成右下降转弯。

无动力右下降转弯

除了下降转弯的主要仪表外，所有仪表都被涂黑了（图6）。假设您刚进入无动力右下降转弯，正在开始三步扫描的步骤2。开始径向扫描主要仪表。调整AI上的姿态，以100节精确下降，标准转弯率。


图6

以下是您用来径向扫描主要仪表的序列。

  1. 从AI开始，径向扫描空速表。
  2. 返回AI，并调整俯仰（如果需要）以保持100节。
  3. 径向扫描转弯协调器。
  4. 返回AI，并调整坡度（如果需要）以保持标准转弯率。（没有理由径向扫描转速表，因为您已将油门减少到飞行慢车。）
  5. 继续径向扫描空速表和转弯协调器，进行小姿态修正，直到飞机在100节建立标准转弯率右下降转弯。
